## Confidence Ratings (1-5 Scale)

For **each tool above**, respondents rate their confidence in these 5 categories:

| Category | Description |
|----------|-------------|
| Awareness | General knowledge of the tool |
| Know How to Access | Can log in and navigate to the tool |
| Know How to Setup | Can configure the tool for use |
| Usage | Proficiency in day-to-day use |
| Need Training | Self-assessed need for additional training |

### Rating Scale

| Rating | Meaning |
|--------|---------|
| 1 | No Knowledge |
| 2 | Basic Awareness |
| 3 | Moderate |
| 4 | Proficient |
| 5 | Expert Level |

---

## Score Calculations

### Capability Score
```
(Number of "Yes" answers / 33) × 100 = Capability %
```

### Confidence Score
```
Average of all 35 confidence ratings (7 tools × 5 categories)
```
